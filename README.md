# AI StoryForge
AI StoryForge is an interactive story generation application that uses large language models to create dynamic, branching narratives. Users can input a story idea and interact with the AI to progress through a choose-your-own-adventure style experience.

## Overview
This application generates interactive stories where each scene is followed by multiple choices. The user's selection determines how the story continues, creating a dynamic narrative experience powered by generative AI.
The project demonstrates the use of large language models in structured output generation, state management, and interactive UI design using Streamlit.

## Features
- Interactive AI-generated storytelling
- Branching narrative based on user choices
- Theme-based story generation (Sci-Fi, Fantasy, Horror, Mystery, Comedy)
- Session-based memory for story continuity
- Dynamic choice extraction from AI responses
- Restart and reset functionality
- Web-based user interface

## Tech Stack
- Python
- Streamlit
- Groq API (LLaMA models)
- python-dotenv (for local environment management)
